# AAD-HR-Visualizer
This is an interactive H-R Diagram using stars from a database (Gaia) using astroquery, Plotly, and pandas, and written on Jupyter. An H-R diagam is an astronomical graph plotting Luminosity vs Temperature, where Luminosity is in absolute magnitude and Temperature is in Kelvin. Placement on an H-R diagram can depict a star's approximate stellar evolution stage and compare it to other stars.
